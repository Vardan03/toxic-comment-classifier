# Toxic Comment Classification 🧠

A machine learning project focused on detecting and classifying toxic comments using natural language processing (NLP) techniques.

## 📌 Project Overview
This project is based on the Jigsaw Toxic Comment Classification challenge.  
The goal is to build a multi-label classification model that can identify different types of toxicity in online comments.

Each comment may belong to one or more of the following categories:
- toxic
- severe_toxic
- obscene
- threat
- insult
- identity_hate

## 🚀 Features
- Text preprocessing and cleaning pipeline
- Feature extraction using TF-IDF
- Baseline model with Logistic Regression
- Multi-label classification
- Evaluation using ROC-AUC
- Modular and scalable project structure
